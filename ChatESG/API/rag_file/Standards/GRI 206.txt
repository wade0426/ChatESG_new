GRI 206：反競爭行為
簡介 GRI 206：反競爭行為 所涵蓋的揭露項目，提供組織報導有關反競爭行為的衝擊資訊，以及如何管理這些衝擊。 準則的架構： 章節1：包含一個報導要求，提供組織如何管理其反競爭行為相關衝擊的資訊。 章節2：包含一個揭露項目，提供組織反競爭行為相關衝擊的資訊。 詞彙表：包含已有定義之用詞，且在GRI準則內文中劃有底線並可連結至其定義。 參考文獻：詳列開發準則時所使用的政府間官方文件。 簡介中其他部分概述了主題的背景說明、GRI準則的系統以及準則使用的更多資訊。 背景說明 本準則闡述反競爭行為主題，包括反托拉斯和壟斷行為。 反競爭行為指組織或其員工與潛在競爭者進行限制市場競爭或聯手勾結的行為，如固定價格、協調出價、產出限制、地域配額，以及分配客戶、供應商、地理區域或產品線。 反托拉斯和壟斷行為可導致建立進入該行業的障礙，或阻止其他競爭行為，例如不公平的商業行為、濫用市場地位、聯合壟斷、反競爭合併和價格固定。 這些概念已在經濟合作暨發展組織（OECD）的重要文書中提及（詳見參考文獻）。 GRI準則的系統 本準則是GRI永續報導準則（GRI準則）的一部分，幫助組織報導其對經濟、環境和人群（包含人權）的衝擊資訊。 準則系統架構於三個互有關聯的系列準則： 通用準則（GRI 1、GRI 2及GRI 3）： GRI 1：基礎 2021：組織依循報導要求的基礎。 GRI 2：一般揭露 2021：提供報導實務和其他詳細資訊（如活動、治理和政策）。 GRI 3：重大主題 2021：決定重大主題的指引，提供如何管理每個主題的揭露項目。 行業準則：提供組織在決定重大主題時可能相關的資訊。 主題準則：供組織報導其對特定主題的衝擊資訊。
