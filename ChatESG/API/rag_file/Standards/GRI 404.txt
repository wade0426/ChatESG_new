GRI 404：訓練與教育
簡介 GRI 404旨在讓組織報導其訓練和教育相關的影響資訊，並說明如何管理這些影響。本準則的架構如下： 
章節1：包含一個報導要求，說明組織如何管理與訓練和教育相關的影響。 
章節2：包含三個揭露項目，提供組織訓練和教育的影響資訊。 
詞彙表：定義了用於GRI準則的特定詞彙和含義，並提供連結至其定義。 
參考文獻：列出本準則的開發所依據的官方文件。 背景說明 此準則針對訓練和教育進行詳細探討，包括組織在培訓員工、職涯發展檢核、以及就業過渡協助方案方面的方針。組織也可涵蓋員工退休或終止合約的生涯管理措施。此內容與國際勞工組織（ILO）及經濟合作暨發展組織（OECD）的相關文件相符。 使用本準則 不論組織的規模、行業或地理區域，皆可參照此準則報導其訓練與教育的相關影響。當組織將訓練和教育視為重大主題時，應報導以下內容： 管理揭露：組織應使用GRI 3：重大主題 2021的揭露項目3-3報導其如何管理訓練與教育（參見條款1-1）。 揭露項目：針對訓練和教育的特定影響，包括404-1至404-3的揭露項目。 主題管理揭露 依循GRI準則，組織必須詳細報導如何管理其重大主題。當組織決定訓練與教育為其重大主題時，應依據GRI 3：重大主題 2021中的揭露項目3-3進行說明（條款1.1）。
