GRI 413：當地社區衝擊揭露 GRI 413為組織提供指導，幫助其揭露對當地社區產生的經濟、社會和環境影響。透過揭露當地社區衝擊，組織可展示其如何管理、預防和減緩對社區可能的負面影響，以及其與社區的合作情況。 1. 主題管理揭露 組織若決定當地社區為其重大主題之一，應根據GRI 3：重大主題 2021中的揭露項目3-3進行揭露，並應包括以下內容： 利害關係人鑑別及議合流程：描述組織如何鑑別與當地社區有關的利害關係人及進行議合的方法。 弱勢群體識別：具體說明當地社區中的弱勢群體，例如因年齡、原住民身份、種族或移民身份而定義的特定群體。 社區特定關注的權利：描述當地社區中是否存在需特別關注的集體或個人權利。 與社區特有利害關係人團體議合：解釋組織如何與當地的特有利害關係人團體合作，如原住民群體。 處理風險及衝擊的方式：說明組織如何管理當地社區風險，或透過支持獨立第三方協助議合，處理衝擊。 2. 主題揭露 組織應揭露其針對當地社區的影響，具體說明如下： 利害關係人之權利與國際公約：根據國際人權宣言、公民權利和政治權利國際公約、經濟社會和文化權利國際公約等文件，組織應尊重並維護當地社區所享有的個人與集體權利。 透過這些指引，組織能更加透明地展示其如何評估和處理對當地社區的影響，增進社區和組織間的相互理解與信任。GRI 413-1：當地社區議合、衝擊評估和發展計畫的營運活動 **揭露項目目標**
