GRI 3：重大主題


使用GRI 3:

 - 依循GRI準則報告的組織必須決定其重大主題並報告GRI 3 中的所有揭露項目。
 - 針對每個重大主題，都必須報告揭露項目 3-3。
 - 省略理由僅可用於揭露項目 3-3。
 - 若無法符合要求，須在GRI內容索引表中說明省略理由 (參考GRI 1:基礎 2021中的要求6)。
 - 若特定子項目不存在，可說明此特例並解釋原因或發展計畫。
 - 已公開報告的資訊無需在獨立報告中重複，只需在GRI內容索引表中提供參考資料即可。


決定重大主題的指引 (四個步驟)

* 這四個步驟提供指引，而非要求。
* 過程應系統化、有文件記錄、可複製，並在每個報導期間保持一致。
* 任何變更應記錄其理由和影響。
* 最高治理單位（或高階管理階層）應監督此流程並核准重大主題。

步驟1. 了解組織脈絡:  建立組織活動、商業關係、永續性脈絡和利害關係人概況的整體總覽。  需考量所有受控或擁有利益的實體（包含少數股權），以及組織內相關部門和職能（溝通、人力資源、投資者關係、法律、行銷、採購、產品開發）的資訊。  GRI行業準則亦可提供協助。

步驟2. 鑑別實際及潛在衝擊: 在組織活動和商業關係中鑑別對經濟、環境和人群（含人權）的實際和潛在衝擊（正面和負面）。  可利用組織內外部評估、法律審查、稽核、申訴機制、風險管理系統和外部來源資訊等。

步驟3. 評估衝擊的顯著程度: 評估已鑑別衝擊的顯著程度，以排定優先順序。  此評估包含定量和定性分析，可能需要主觀判斷，並宜諮詢利害關係人和專家。

 - 負面衝擊: 嚴重性（規模、範疇、無法補救的特徵）和可能性。 人權衝擊，嚴重性優先於可能性。
 - 正面衝擊: 規模、範疇和可能性。

步驟4. 排定最顯著衝擊的優先報導順序:  根據顯著性排定衝擊優先順序，並定義門檻決定要報導的主題。  衝擊宜以主題分組，並參考GRI主題準則和GRI行業準則。  衝擊的顯著性是決定主題是否重大的唯一標準。  應分別排定負面和正面衝擊的優先順序。
