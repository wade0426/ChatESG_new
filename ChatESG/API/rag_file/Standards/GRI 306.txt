GRI 306：廢棄物
簡介 GRI 306：廢棄物 所涵蓋的揭露項目，提供組織報導有關廢棄物的衝擊資訊及其管理方式。 準則架構： 章節1：包含兩個報導要求，說明組織如何管理廢棄物相關的衝擊。 章節2：包含三個揭露項目，提供組織廢棄物相關衝擊的具體資訊。 詞彙表：提供定義的用詞，劃有底線並可連結至其定義。 參考文獻：列出開發準則時使用的政府間官方文件。 背景說明 本準則聚焦於廢棄物主題，指出廢棄物可由組織的生產和服務活動產生，也可能由上游（供應商）或下游（消費者）實體在處理組織的物料、產品或服務時產生。 未妥善管理的廢棄物可能對環境和健康造成負面衝擊，並導致資源浪費。資源經焚化或掩埋後，將無法被重複利用，加速其消耗。 聯合國可持續發展目標（SDG）12中，強調了責任消費與生產，鼓勵組織採取無害的廢棄物管理，並推動廢棄物的再利用和減少。 使用本準則 本準則適用於任何規模、行業、地區的組織，以及管理其他組織廢棄物的機構（如公營和民營的廢棄物管理機構）。 若組織確定廢棄物為重大主題，應報導以下揭露項目： GRI 3：重大主題 2021中的揭露項目3-3（參閱條款1-1）。 本主題準則中與廢棄物衝擊相關的揭露項目（306-1至306-5）。 如有需要，可參考GRI 301：物料 2016中的相關揭露項目。 報導例外與說明 若無法滿足特定揭露項目（如機密信息或法律限制），組織應在GRI內容索引表中指明未達成的項目並提供省略原因。 若揭露項目要求的資訊（如政策、流程）不存在，組織應說明原因或描述發展計畫。 獨立報告 若組織已在其他地方（如官網或年報）公開相關資訊，無需在永續報告中重複，可在GRI內容索引表中提供指向該資訊的連結或頁碼。要求報導組織應使用GRI 3：重大主題 2021中的揭露項目3-3來報導組織如何管理廢棄物。
