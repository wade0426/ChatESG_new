GRI 415：公共政策 2016 - 主題簡介 背景說明 本準則探討組織在公共政策方面的參與，包含活動（如遊說）以及向政黨、政治家或相關事業的財務或實物捐贈。組織可以積極參與公共政策的發展，但也可能帶來與貪腐、賄賂和不當影響等相關的風險。 揭露要求 依循GRI準則進行報導的組織若將公共政策列為重大主題，應報導以下揭露項目： GRI 3：重大主題 2021中的揭露項目3-3（依條款1.1）。 主題揭露 使用 GRI 3 的揭露項目3-3，說明組織如何管理公共政策的相關影響。 建議揭露 組織宜提供： 組織參與的公共政策發展與遊說重點議題。 組織在此類議題上的立場，並說明任何與既定政策、目標或其它公共立場之間的差異。 報導指南 若無法完全滿足揭露項目或其要求，應在內容索引表中指明無法符合的項目，並提供合理說明（參閱GRI 1：基礎 2021中的要求6）。 背景重要性 這些揭露項目能幫助利益相關者瞭解組織對公共政策的影響，並加強組織對於透明、負責任的政策參與之承諾。
